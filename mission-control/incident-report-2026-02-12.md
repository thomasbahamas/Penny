## What is HTTP 406?
**406 Not Acceptable** = Server cannot produce a response matching the request's Accept headers.

Common causes:
1. **Content negotiation failure** â€” Client requests JSON, server only serves HTML (or vice versa)
2. **Reverse proxy misconfiguration** â€” Cloudflare/nginx blocking based on headers
3. **Missing API version header** â€” Server expects version, client doesn't send it
4. **Bot protection** â€” WAF blocking automated requests
